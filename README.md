# FHQ Booking System

Full-stack barber shop booking platform built with Next.js and PostgreSQL.

## Tech Stack

Next.js  
React  
TailwindCSS  
Prisma ORM  
PostgreSQL (Neon)  

## Integrations

Google Calendar API  
Resend Email API  

## Features

- Customer appointment booking
- Real-time availability selection
- Google Calendar event creation
- Automated email confirmations
- Admin appointment management

## Architecture

Frontend: Next.js + React  
Backend: API routes with Prisma ORM  
Database: PostgreSQL hosted on Neon
